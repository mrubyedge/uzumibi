# Deploying

### Cloudflare Workers

Make sure you have a Cloudflare account and `wrangler` configured:

```bash
npx wrangler login
```

Deploy your application:

```bash
npx wrangler deploy
```

### Fastly Compute

Deploy to Fastly:

```bash
fastly compute deploy
```

### Spin (Fermyon Cloud)

Deploy to Fermyon Cloud:

```bash
spin deploy
```
